# RBI Balance Sheet: A Comprehensive Time Series and Statistical Analysis of Reserve Money, Assets, and Liabilities

This project analyzes the Reserve Bank of India's weekly balance sheet data, with emphasis on reserve money, its asset-side drivers, liabilities, unusual monetary events, and forecasting.

## Project Contents

- `data/raw/RBI_Data.xlsx`: original workbook supplied for the project.
- `data/processed/cleaned_data.csv`: cleaned weekly dataset with calendar, growth, rolling, index, z-score, and contribution features.
- `notebooks/`: 10 professional notebooks covering data understanding through final insights.
- `src/rbi_utils.py`: reusable cleaning and feature engineering functions.
- `dashboard/`: Power BI build guide and dashboard-ready CSV references.
- `reports/`: executive summary and final report drafts.
- `images/`: output folder for charts generated by notebooks.

## Dataset Snapshot

- Frequency: weekly
- Observations: 1,007
- Date range: 2001-07-06 to 2020-08-21
- Unit: Rupees crore
- Main variables: reserve money, currency in circulation, bankers' deposits, claims on government, claims on banks, net foreign exchange assets, government currency liabilities, and net non-monetary liabilities.

## Recommended Workflow

1. Create a Python environment and install dependencies: `pip install -r requirements.txt`.
2. Open the notebooks in order from `01_Data_Understanding.ipynb` to `10_Final_Report_and_Insights.ipynb`.
3. Run `02_Data_Cleaning_and_Feature_Engineering.ipynb` if the raw workbook changes.
4. Use the CSV files in `data/processed/` to build the Power BI dashboard described in `dashboard/PowerBI_Dashboard_Guide.md`.
